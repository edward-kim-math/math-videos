Which of these equations is the equation of the line going through (2,0) and (0,4)? A, x + 2y = 4. B, 2x + y = 4. C, -2x + y = 4. D, 2x - y = 4. E, x - 2y = 4.
We can check which equation is correct by plugging in the coordinates of the two points. That is, both points need to satisfy the equation that is relevant.
Let's check the first equation x plus 2y equals 4 with the points (2,0) and (0,4).
Plugging in the point (2,0) into the equation x + 2y = 4, we get 2 + 2(0) = 4, which is false since the left side simplifies to 2 and the right side simplifies to 4. We don't bother checking the other point.
Let's check the second equation 2x plus y equals 4 with the points (2,0) and (0,4).
Plugging in the point (2,0) into the equation 2x + y = 4, we get 2(2) + 0 = 4, which is true, since both sides simplify to 4.
Plugging in the point (0,4) into the equation 2x + y = 4, we get 2(0) + 4 = 4, which is true, since both sides simplify to 4.
So, equation B, which is 2 x plus y equals 4, is the correct equation. This equation goes through both the point (2,0) and the point (0,4).
