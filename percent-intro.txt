Percent change reflects the change between an old value and a new value relative to the old value.
The formula for percent change is new value minus old value divided by old value, all multiplied by 100.
Let's look at this example. The price of a burger increased from 2 dollars and 50 cents in 2000 to 4 dollars in 2015. What was the percent increase?
To find the percent increase, we can use the percent change formula. The new value is 4 dollars, and the old value is 2 dollars and 50 cents. So we have 4.00 minus 2.50 divided by 2.50, all multiplied by 100.
Note that we are calculating a percent increase, so we are using the new value minus the old value. If we were calculating a percent decrease, we would use the old value minus the new value.
Simplifying the fraction, we have 1.50 divided by 2.50, which is 0.60, which needs to be multiplied by 100, giving a 60 percent increase.
